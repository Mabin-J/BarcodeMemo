BarcodeMemo
===========

Simple tool for scaning barcode and collecting result.
Result is Auto-saving and It can be shared via SMS, E-Mail, Cloud Storage, SNS, etc.

Zxing-2.2 is needed.
(https://code.google.com/p/zxing/)

Feature
-------
- Scanning Barcode
- Continuous Scanning
- Auto-saving
- Undo/Redo
- Share to SMS, E-Mail, etc
